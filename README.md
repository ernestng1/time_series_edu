# Time Series Educational App

A modern, interactive educational web application built with React, Vite, Tailwind CSS, and Recharts. This application is designed to teach fundamental time series concepts including Level, Trend, Seasonality, Noise, and the differences between Additive and Multiplicative models.

## Features

- **Interactive Graphing:** Real-time generation of time series charts as you adjust parameters.
- **Progressive Disclosure:** An intuitive left sidebar utilizing an accordion pattern to prevent visual clutter.
- **Educational Tooltips:** Hover over parameters to understand their statistical meaning.
- **Quiz Mode:** Test your knowledge of additive vs. multiplicative time series patterns.
- **Smooth Animations:** Built with Framer Motion for elegant layout transitions.

## Quick Start

### Installation

1. Ensure you have Node.js installed.
2. Clone this repository or download the source code.
3. Open your terminal in the project directory and run:

```bash
npm install
```

### Running Locally

To start the Vite development server:

```bash
npm run dev
```
Then navigate to the URL provided in your terminal (usually `http://localhost:5173`).

### Building for Production

To create a production-ready build:

```bash
npm run build
```
The compiled files will be located in the `dist` folder, ready to be deployed to GitHub Pages or any static file host.
